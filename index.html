使用说明 Duper 战场编辑器

点格子 = 编辑该地块：归属、地块等级、士兵/坦克/飞机加减、建筑、是否在地图中（可移出以拼出真实地图形状）。

底部三个模式键：
🦶 移动 — 点你的地面兵格 → 沿己方territory流动高亮可达格 → 点目标格移动；同兵种合并、移动力取两者最低（士兵1/坦克5/飞机7）。
⚔️ 攻击 — 点你的进攻格 → 相邻敌/中立目标，弹出 AP/DP，点"打这一仗"占领/扣兵改棋盘。
⎘ 复制为新变化 — 基于当前局面分叉一个新变化继续推演；↶ 撤销上一步。

图标：👣士兵(战力1/移动1) · 🛡️坦克(3/5,可支援) · ✈️飞机(4/7) · ⌂基地 · ▣堡垒(7) · ✦中心 · ⛨该格防御力DP。

队伍色：青=玩家1(庄家) 绿=2 蓝=3 红=4 黄=5。

战斗：AP>DP 占领。DP = 守军 + 建筑 + 地块Buff + 钥匙(防守方持进攻方颜色钥匙每把+2) + 相邻同色坦克支援；NPC格按等级计防御。

存档：自动存到本机浏览器（换设备不通用）。☰ 变化列表里可切换/新建战场、改名/删变化、导入地图编辑器的 JSON 当底图。

导入底图 粘贴 JSON

把地图编辑器导出的 JSON 粘到下面（支持 {cells,keys} 或 2D 数组；字段短名 e/o/l/b/bl/g/gc/a/ac 或长名 enabled/owner/… 都识别）。导入会替换当前局面的棋盘。
